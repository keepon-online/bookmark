# 图标说明

需要添加扩展图标文件到 `public/icons/` 目录：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

建议使用 SVG 设计并导出为 PNG 格式。

图标设计建议：
- 主题：书签、智能/AI、收藏
- 风格：简洁、现代、扁平化
- 主色调：蓝色 (#3b82f6)
- 可选元素：书签图标 + 星星/心形/AI 芯片装饰

可以使用以下工具创建图标：
- Figma / Adobe Illustrator
- Canva
- 在线工具：https://www.favicon-generator.org/

注意：图标文件缺失不影响功能，但会在扩展列表中显示默认图标。
